Fundamentals of
Database Systems

This course develops the knowledge and practical skills needed to organize data, design relational databases, write and evaluate SQL queries, protect and recover database systems, and connect a MySQL database to a Java Swing application through JDBC.

From database foundations to a working CRUD application.
