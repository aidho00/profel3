# PROFEL 3 — Fundamentals of Database Systems

GitHub Pages-ready class materials with 36 lessons grouped into PRELIM, MIDTERM, SEMI FINAL, and FINAL.

## Deployment

1. Upload all files and folders in this directory to the root of a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Choose **Deploy from a branch**, select `main`, and use `/ (root)`.
4. Missing lesson HTML files automatically appear as locked on the landing page.
